 # Model Description

This model is a warm-started BERT2BERT (small) model fine-tuned on the CNN/Dailymail summarization dataset.

The model achieves a 17.37 ROUGE-2 score on CNN/Dailymail's test dataset.

# How API works

- Click try it out under 'POST' method. 
- Past the text as a JSON string
- Click execute
- Output will be a summarized JSON text

The response will be a JSON with answer.
